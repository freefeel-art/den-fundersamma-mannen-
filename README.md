# Den Fundersamma Mannen

## En man som funderar

Den Fundersamma Mannen är ett pågående projekt.

Inte en blogg.
Inte en kurs.
Inte ett företag.

Bara en plats där tankar får leva vidare.

Här samlas reflektioner om:

- livet
- nykterhet och förändring
- företagande
- nätverksmarknadsföring
- artificiell intelligens
- personlig utveckling
- Finland och Sverige
- människan och hennes val

Projektet växer organiskt.

Vissa texter börjar som samtal.
Andra börjar som frågor.

Några blir artiklar.

Andra blir något helt annat.

---

> "Det börjar ofta med en fundering."

/ Jarmo Halonen
